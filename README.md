# [Большой список с выбором и сортировкой](https://digital-solution-back.netlify.app/)

Полноценный пример приложения с 1 000 000 элементов, двумя окнами выбора, Drag&Drop‑сортировкой, инфинити‑скроллом и очередью запросов с батчингом.

## Технологии

- **Backend**: Node.js, Express (in‑memory хранилище, без БД)
- **Frontend**: React 18, TypeScript, Vite

## Запуск проекта

### 1. Установка зависимостей

```bash
cd "d:\Projects\Цифровые решения"

cd backend
npm install

cd ../frontend
npm install
```

### 2. Запуск backend

```bash
cd "d:\Projects\Цифровые решения\backend"
npm run start
# сервер поднимется на http://localhost:4000
```

### 3. Запуск frontend

```bash
cd "d:\Projects\Цифровые решения\frontend"
npm run dev
# откройте в браузере http://localhost:3000
```

## Архитектура и требования

- **Левое окно**: показывает все элементы, кроме выбранных; фильтр по ID; инфинити‑скролл (порции по 20 элементов); добавление новых ID.
- **Правое окно**: показывает выбранные элементы; фильтр по ID; Drag&Drop‑сортировка (включая отфильтрованный список); инфинити‑скролл (порции по 20 элементов).
- **Сохранение состояния**: выбор и порядок выбранных хранятся на сервере в памяти процесса и сохраняются между обновлениями страницы (до перезапуска сервера); поиск не сохраняется.
- **Очередь запросов**:
  - дедупликация повторяющихся запросов;
  - батчинг: добавление элементов — раз в 10 секунд; чтение и изменение данных — раз в 1 секунду.
- **Учётные записи не используются**: данные общие для всех посетителей, пока живёт процесс Node.js.
